Pembahasan Soal 1

1.
Soal: $f(x) = 2x + 3, g(x) = x - 5$. Tentukan $(f + g)(x)$.
Jawaban: A. $3x - 2$

Pembahasan:
$(f + g)(x) = f(x) + g(x)$
$= (2x + 3) + (x - 5)$
$= 2x + x + 3 - 5$
$= 3x - 2$

Pembahasan Soal 8

8.
Soal: Invers dari $f(x) = \frac{2x - 3}{x + 4}$.
Jawaban: B. $\frac{-4x - 3}{x - 2}, x \neq 2$

Pembahasan:
Rumus cepat invers pecahan linear: $f(x) = \frac{ax + b}{cx + d} \Rightarrow f^{-1}(x) = \frac{-dx + b}{cx - a}$.
Disini $a=2, b=-3, c=1, d=4$.
$$ f^{-1}(x) = \frac{-4x - 3}{1x - 2} = \frac{-4x - 3}{x - 2} $$

Pembahasan Soal 13

13.
Soal: $(f \circ g)(x) = 2x^2 - 4x + 6, g(x) = x - 1$. Tentukan $f(x)$.
Jawaban: A. $2x^2 + 4$

Pembahasan:
Misal $u = x - 1 \Rightarrow x = u + 1$.
$f(u) = 2(u+1)^2 - 4(u+1) + 6$
$= 2(u^2 + 2u + 1) - 4u - 4 + 6$
$= 2u^2 + 4u + 2 - 4u + 2$
$= 2u^2 + 4 \Rightarrow f(x) = 2x^2 + 4$.

17.
Soal: Nilai $(g \circ f)^{-1}(2)$ jika $f(x) = 3x - 1, g(x) = \frac{x}{x+1}$.
Jawaban: A. $-\frac{1}{3}$

Pembahasan:
Misal $(g \circ f)^{-1}(2) = a \Rightarrow (g \circ f)(a) = 2$.
$g(f(a)) = 2 \Rightarrow \frac{f(a)}{f(a)+1} = 2$.
$f(a) = 2f(a) + 2 \Rightarrow f(a) = -2$.
$3a - 1 = -2 \Rightarrow 3a = -1 \Rightarrow a = -1/3$.

18.
Soal: Cari $p$ jika $(f \circ g)(x) = (g \circ f)(x)$ untuk $f(x)=2x+p, g(x)=3x+120$.

$240 + p = 3p + 120 \Rightarrow 120 = 2p \Rightarrow p = 60$.

Pembahasan Soal 19

19.
Soal: Invers dari $g \circ f$ jika $f(x)=x+2, g(x)=\frac{2x}{x-1}$.
Jawaban: A. $\frac{4 - x}{x - 2}$

Pembahasan:
$(g \circ f)(x) = g(x+2) = \frac{2(x+2)}{(x+2)-1} = \frac{2x+4}{x+1}$.
Gunakan rumus invers $(ax+b)/(cx+d) \to (-dx+b)/(cx-a)$.
Disini $a=2, b=4, c=1, d=1$.
Invers = $\frac{-x+4}{x-2} = \frac{4-x}{x-2}$.

Pembahasan Soal 20

20.
Soal: Domain $(f \circ g)^{-1}(x)$ jika $f(x)=1/x, g(x)=x+1$.
Jawaban: A. $x \neq 0$

Pembahasan:
$(f \circ g)(x) = f(x+1) = \frac{1}{x+1}$.
Misal $y = \frac{1}{x+1} \Rightarrow x+1 = \frac{1}{y} \Rightarrow x = \frac{1-y}{y}$.
Maka $(f \circ g)^{-1}(x) = \frac{1-x}{x}$.
Syarat pecahan: penyebut $\neq 0$, jadi $x \neq 0$.
